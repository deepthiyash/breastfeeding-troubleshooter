# LatchCare Assistant

A simple Streamlit MVP for breastfeeding education and troubleshooting.

## What it does

Users select a breastfeeding concern, answer a few safety questions, and receive general educational guidance plus red-flag warnings.

## Important disclaimer

This app is for education only. It does not diagnose, treat, or replace care from a doctor, pediatrician, OB/GYN, or certified lactation consultant.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud

1. Create a GitHub repository.
2. Upload these files.
3. Go to Streamlit Community Cloud.
4. Connect your GitHub repository.
5. Set the main file as:

```text
app.py
```
